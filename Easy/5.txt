Quick help - you have to use one of bitwise operators

Input 
An integer num 

Output 
An integer result, which is the transformed value of num 

Constraints 
possible solutions
 
Example 

Input 
5 

Output 
-6

Input 
0

Output 
-1

Input 
-3

Output 
2
