You are given three numbers a, b and c. 
Starting from 0, compute the total: - 
If the number is even, add it to the total. - 
If the number is odd, subtract it from the total. 

Input 
Line 1: An integer a. 
Line 2: An integer b. 
Line 3: An integer c. 

Output 
The total following the given rule. 

Constraints 
0 ≤ a ≤ 1000 
0 ≤ b ≤ 1000 
0 ≤ c ≤ 1000 

Example 
Input 
3 
2 
4 

Output 
3



-------Only see the solution after putting your efforts into it---------

//reading inputs
const a = parseInt(readline());
const b = parseInt(readline());
const c = parseInt(readline());

let total = 0;
[a,b,c].forEach((num)=> {
        if(num%2===0) total += num;
        else total -= num;
})

console.log(total);
